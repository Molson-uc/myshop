# Myshop
## Introdaction
This Django project is fully operational online shop.
## Features
- Python 3.10
- Django
- payment system using Braintree
- send email
- celery
- Sqlite3
## Setup
'''
https://github.com/Molson-uc/myshop.git
python3 -m venv venv
source venv/bin/activate
cd myshop
pip install -r requirements.txt
python manage.py runserver
'''
